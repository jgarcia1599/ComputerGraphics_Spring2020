These are file added by me (saurabh).

Trackball.js contains implementation of a virtual trackball.

Camera.js contains the implementation of a camera object.
Camera - v1.js contains a first version of Camera.js without camera movements.

Utils.js contains some utility functions.
Utils -v1.js contains a first version of Utils.js without trackballWorldMatrix().

Object.js contains object definition.
Object - v1.js: first version of Object.js without normal mapping.
